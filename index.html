🔗 Kết nối GitHub để lưu data

Data lưu thẳng vào 1 file JSON trên GitHub repo của bạn — miễn phí, không cần server, mobile & desktop đều dùng được.

1
Vào github.com → đăng nhập → tạo repo tên k45-data, chọn Public
2
Trong repo đó tạo file data.json với nội dung {"teachers":[],"selected":{}} → Commit
3
Vào Settings → Developer settings → Personal access tokens → Fine-grained → tạo token, cấp quyền Contents: Read & Write cho repo k45-data
4
Dán thông tin vào dưới, bấm Bắt đầu

Token chỉ lưu trong máy bạn (localStorage), không ai khác thấy được.
Chỉ người setup mới cần làm bước này — người khác dùng link bình thường.
